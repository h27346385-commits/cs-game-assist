### 技术架构

```
Electron 40.6
├── 主进程 (Node.js)
│   ├── 文件对话框
│   ├── 系统API
│   └── 窗口管理
│

```

### 系统要求

- Windows 10/11 64位
- 4GB 内存
- 100MB 磁盘空间

### 注意事项

⚠️ 本版本为 **MVP Demo**，使用**模拟数据**

真实功能将在后续版本添加：
- Demo文件解析
- 视频生成 (HLAE+FFmpeg)
- 平台数据同步
- 本地数据库存储

### 开发调试

如需修改代码：

```bash
# 1. 安装依赖
npm install

# 2. 启动开发模式
npm run electron:dev

# 3. 重新打包
npm run electron:build:portable
```
